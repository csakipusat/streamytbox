# StreamYtbox

**StreamYtbox** is a lightweight and reliable YouTube stream management platform  
designed to manage, monitor, and automate live streams with ease.

## Features
- Manage multiple YouTube live streams
- Schedule and automate streams
- Monitor status and logs
- Secure configuration
- Optimized for Linux

## Installation (APT)
```bash
sudo apt install streamytbox
```
